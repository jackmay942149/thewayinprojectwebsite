Mission

As the founder, Jack, a third-year student at QUT studying a Bachelor of Games and Interactive Environments, I am excited to introduce The Way In Project.

Our mission is to empower and connect entry-level entertainment game and film developers in Queensland.

Inspired by my own experiences, I created this project to address the challenge of discovering valuable networking events and resources in the local industry.

While Queensland boasts a wealth of talent and opportunities, staying informed about upcoming events can be challenging.

The Way In Project aims to bridge this gap by providing a centralized platform for developers to access comprehensive resources and stay informed about impactful events.

We are dedicated to cultivating a supportive ecosystem that propels emerging talent into successful careers within the entertainment industry.

Join us on this journey as we shape the future of Queensland's entertainment landscape, one connection at a time.

Featured Event

Brisbane International Game Developers Association's Brisbane Global Game Jam

January 26th 4:00pm - January 28th 5:00pm

QUT Foundry - 2 George Street Brisbane City, QLD 4000

The Global Game Jam® (GGJ) is the world’s largest game jam event (game creation) taking place around the world at physical (and virtual) locations over the same weekend! GGJ is a seriously fun event with big community vibes. Joining a site is about an adventure in game making (digital, analogue, whatever) and creativity. The Brisbane site for Global Game Jam is Back! We will be running a 48 hour site for the main weekend of the Global Game Jam from Friday January 26th - Sunday January 28th.
