Brisbane International Game Developers Association's Brisbane Global Game Jam

January 26th 4:00pm - January 28th 5:00pm

QUT Foundry - 2 George Street Brisbane City, QLD 4000

The Global Game Jam® (GGJ) is the world’s largest game jam event (game creation) taking place around the world at physical (and virtual) locations over the same weekend! GGJ is a seriously fun event with big community vibes. Joining a site is about an adventure in game making (digital, analogue, whatever) and creativity. The Brisbane site for Global Game Jam is Back! We will be running a 48 hour site for the main weekend of the Global Game Jam from Friday January 26th - Sunday January 28th.
